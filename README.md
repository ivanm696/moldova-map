# 🗺️ Харта Интерактивэ а Молдовей

Интерактивная карта Молдовы на молдавском языке (кириллица, Lumina Standard).

**Live Demo:** [ivanm696.github.io/moldova-map](https://ivanm696.github.io/moldova-map)

## Функции

- 🖱️ Кликни на район — увидишь название, население, площадь на кириллице
- 🔍 Поиск по районам (молдавский / румынский / русский)
- 📊 Статистика: население, площадь, плотность
- 🌙 Тёмная тема
- 33 района Молдовы с данными

## Стек

React 18 + TypeScript + Vite + CSS

## Запуск

```bash
git clone https://github.com/ivanm696/moldova-map.git
cd moldova-map
npm install
npm run dev
```

## Язык

Все названия — молдавский язык, кириллица (Лумина Стандард, ISO 639: mol)
